## Backend Shared Logic

This folder stores shared business logic copied from `apps/web` and `apps/mobile`.

Current modules:
- `src/shared/booking/slot-core.ts`
- `src/shared/booking/email-payloads.ts`
- `src/shared/booking/status.ts`
- `src/shared/time/malaysia.ts`

No files in `apps/web` or `apps/mobile` are modified by these additions.
